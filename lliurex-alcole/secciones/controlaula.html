Controlant l'aula amb l'Epoptes

Programari lliure creat i mantingut per un equip de desenvolupadors grec

  • Lloc web: http://www.epoptes.org

Què em permet fer?

  • Maquinari

  • Demostracions

  • Executar

  • Restringir

1. Maquinari

  • Iniciar

    Arranca tots els clients seleccionats.

  • Tancar la sessió

    Tanca la sessió dels usuaris amb sessió activa.

  • Reiniciar

    Reinicia els clients seleccionats.

  • Apagar

    Apaga els clients seleccionats.

Nota: Cal afegir tots els equips a un grup per a poder realitzar aquestes accions.

2. Demostracions

  • Veure l'alumnat

    Obri una finestra que mostra el que fa l'alumne.

  • Controlar l'alumnat

    Permet veure el que fa l'alumne i interactuar-hi.

  • Demostració

    Mostra en els clients el que veiem al nostre monitor.

  • Demostració (finestra)

    Mostra als clients el que fem en una finestra.

  • Parar les demostracions

    Acaba les demostracions actives.

3. Executar

  • Executar

    Executa una ordre/aplicació en els clients.

  • Enviar un missatge

    Mostra un missatge en l'equip client.

  • Obrir un terminal

    Obri un terminal en els clients per a executar ordres.

4. Restringir

  • Bloquejar

    Bloqueja l'equip del client.

  • Desbloquejar la pantalla

    Elimina el bloqueig de pantalla en els equips.

  • Bloquejar internet

    Restringeix l'accés a internet en els clients.

  • Desbloquejar internet

    Elimina la restricció d'internet en els clients.

  • Desactivar el so

    Inhabilita el so en els clients.

  • Activar el so

    Habilita el so en els clients.
